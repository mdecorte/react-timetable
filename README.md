# React Timetable
Create timetable from Festival data JSON

## Festival Timetable

### What is it:
Create festival timetable from JSON data and be able to create (and export?) a personalized Timetable.

### Techniques:
- React
- Webpack
- service worker

### Work it:
Install dependencies:  
`$ yarn`  

Run server:  
`$ yarn start`
